# 🛍️ Awesome – LimeRoad Clone

A fully responsive clone of **LimeRoad**, developed as an individual project in **5 days**. The application recreates the core shopping experience of India's leading fashion marketplace, allowing users to browse products, view details, authenticate, and manage their shopping cart through an intuitive and responsive interface.

> **Note:** This project was built for educational purposes only and is not affiliated with LimeRoad.

# 📖 About The Project

LimeRoad is one of India's most popular online fashion marketplaces for women, men, and kids. This project replicates its core shopping experience using React and Chakra UI while following a component-based architecture.

The objective of this project was to strengthen frontend development skills by building a complete e-commerce application featuring routing, reusable components, state management, authentication, and responsive layouts.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Anubhav0391/awesome.git
```

Navigate to the project

```bash
cd awesome
```

Install dependencies

```bash
npm install
```

Run React application

```bash
npm start
```

Run JSON Server

```bash
npx json-server --watch db.json --port 8080
```
